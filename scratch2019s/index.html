Advanced Scratch Programming

For 2nd Grade to 6th Grader, who have some scratch programming experience. We will cover some advanced Scratch programming topics:

  • Games: Platformer, Flappy Bird
  • Simulation: Snow, fireworks
  • Geometry
  • Advanced Topics: function, clone, sensor, etc

Requirements

  1. Some Scratch Programming experience.
  2. Bring your own computer.
